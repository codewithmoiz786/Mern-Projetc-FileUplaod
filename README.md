# MERN File Upload - ImageKit + Express

Simple file upload project using Express, Multer, ImageKit and vanilla HTML/JS frontend.

## Features
- 📤 Image upload to ImageKit CDN
- 💾 Store image URL + caption in MongoDB 
- 🖼️ Display all uploaded images in grid
- 🔒 .env keys protected with .gitignore
- 🌐 CORS enabled for frontend

## Tech Stack
**Backend:** Node.js, Express, Multer, ImageKit SDK, MongoDB, Mongoose  
**Frontend:** HTML5, CSS3, Vanilla JavaScript
